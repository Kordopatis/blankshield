The first link below is vulnerable to reverse tabnabbing since it uses target="_blank". Try clicking the link normally, or with a modifier key (shift, meta, etc), but without using a right click context menu. You'll be taken to a new tab, and this page will redirect to GitHub. The new tab is able to redirect this page using window.opener.location.assign(). The second link, however, is protected by using blankshield(), the function provided by this library.
